# TextExcel
TextExcel Project for AP CS.

TextExcel behaves like a spreadsheet. It can handle strings, numeric data, overflow text, and formulas. 

How to input commands:

	Assigning a cell:
		C3 = "Hello World"
		A1 = 123
		B2 = ( 3.2 + 12 * 14 )
		D5 = 35%
		E4 = ( A1 + 3.4 )
		F6 = ( Sum A1-C3 )
		C2 = ( avg B2-D7 )
		
	Clearing: 
		Clear B3
		Clear //For entire sheet
